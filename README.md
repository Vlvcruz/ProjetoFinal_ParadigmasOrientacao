Projeto desenvolvido com finalidade acadêmica para a conclusão do Segundo Bimestre do curso de Bacharelado em Sistemas de Informação da Universidade de Mogi das Cruzes (UMC).
Docente responsável: Paulo José de Carlo Almeida.

Observações para execução do programa:

- Recomenda-se utilizar o Visual Studio Code para garantir o bom funcionamento da aplicação.
- É necessário instalar e utilizar a extensão Thunder Client (disponível dentro do próprio VS Code) para realizar operações como INSERT, DELETE, entre outras, por meio de requisições JSON.
- É indispensável a instalação do JDK 17 para que a aplicação seja executada corretamente.
